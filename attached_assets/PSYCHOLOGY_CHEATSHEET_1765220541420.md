## THE 4 CORE FRAMEWORKS

### 1️⃣ NEUROSCIENCE & NLP

**Brain Decision Hierarchy:**
```
Reptilian (survival) → Limbic (emotion) → Neocortex (logic)
```
**Rule:** ALWAYS trigger emotion BEFORE logic.

**NLP Patterns:**
| Pattern | Example |
|---------|---------|
| Presupposition | "**When** you implement this..." |
| Embedded Command | "You might **find yourself wondering** why you waited" |
| Pattern Interrupt | "I'm not going to pitch you anything" |
| Future Pacing | "**Picture this:** 90 days from now..." |
| Sensory Language | "**Feel** what it's like when..." |

---

### 2️⃣ BEHAVIORAL ECONOMICS

**Loss Aversion (2.5x multiplier):**
```
❌ BAD:  "You could earn $3,000/month more"
✅ GOOD: "You're LOSING $3,000/month right now"
✅ BEST: "That $100/day loss has already cost you $3,000 this month"
```

**The 6 Principles:**

| Principle | Application |
|-----------|-------------|
| **Loss Aversion** | Frame as preventing LOSS, not acquiring gain |
| **Anchoring** | Set HIGH reference first, then reveal price |
| **Social Proof** | "37 restaurants in Colombia use this" (specific > general) |
| **Scarcity** | "I take 3 new clients per month" (must be genuine) |
| **Reciprocity** | Give value BEFORE asking: "I did a quick audit..." |
| **Commitment** | Get small yeses → "Would you agree no-shows are a problem?" |

---

### 3️⃣ IDENTITY TRANSFORMATION

**The Gap = Purchase Motivation**

| Current Identity | → | Aspirational Identity |
|------------------|---|----------------------|
| Overwhelmed, reactive | | Strategic, in control |
| "I can't keep up" | | "My business runs itself" |
| Putting out fires | | Building something |

**Identity Triggers by Vertical:**
```
Restaurant: "You didn't open a restaurant to be a full-time receptionist."
Hotel:      "You built a hotel to create experiences, not answer the same question 50x."
Tour:       "You started this to share adventures, not be buried in logistics."
Charter:    "You bought boats to live the life, not be chained to your phone."
```

**Power Questions:**
- "Is this who you want to be known as?"
- "Is this the operation you dreamed of building?"
- "A year from now, will you be glad you made this move?"
